# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### [21.11.2025]

#### Added
- **ResetUI**: New option to reset all features to their original default values
- **Always-on-top UI**: Function to keep the UI layer above all other interface elements at all times
- **Video feature**: New video functionality integration

---

### [15.11.2025]

#### Changed
- **Floating menus**: Menus can now only be dragged by their title bar for improved control
- **Toggle interaction**: Clickable area now extends from the checkbox to the end of the label text for better UX

#### Fixed
- **Slider mutex**: Only one slider can be active at a time, preventing conflicts

---

### [08.11.2025]

#### Added
- **Cursor guide**: Customizable cursor guide with adjustable properties
- **Draggable ColorPicker**: ColorPicker menu can now be repositioned by dragging

---

### [02.11.2025]

#### Added
- **Optional tooltips**: Tooltips are now optional and can be disabled per component
- **Divider with label**: Dividers now support centered text labels with proper spacing

#### Fixed
- **WatermarkText bug**: Resolved issue where watermark remained visible after being disabled

---

### [25.10.2025]

#### Added
- **Tab icons**: Tabs now support icon integration for enhanced visual identification
- **Mobile button alignment**: Mobile buttons can now be positioned on either left or right side (default: left)

---

### [05.09.2025]

#### Added
- **Slider value display**: ValueText property added to Slider component
- **Tab icons**: Icon support added to Tab component

---

## Legend

- `Added` - New features
- `Changed` - Changes in existing functionality
- `Deprecated` - Soon-to-be removed features
- `Removed` - Removed features
- `Fixed` - Bug fixes
- `Security` - Vulnerability patches
